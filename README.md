# CATTI 视觉化笔译训练工作台 V8 回滚增强版

V8 的目标是修复 V7 过度复杂导致的退化：恢复 V5/V6 的知识密度、真实知识点地图、英语记忆条、RSS 知识流和长难句实验室；同时新增一个离线优先的稳定题库训练台。

## 核心变化
- 恢复并强化：知识点地图、448+知识卡、英语记忆条、长难句、句子下划线分析、搭配记忆。
- 新增：系统讲义模块，按时态、名词、从句、非谓语、被动、信息结构、介词、程度副词、近义词、术语策略串联。
- 新增：离线题库训练台，支持选择题、阅读理解、短句翻译、搭配填空、改错润色。可随机换题、提交评分、显示参考答案。
- 新增：示范讲解题库，强调“为什么这样译”。
- AI 改为可选补充：默认不依赖 API；AI 生成题目成功后会放入训练台顶部。

## 手机端
- 底部导航 + 目录抽屉 + 快速设置。
- 训练台在手机端单列显示，控制区保持在上方，便于换题和提交。

## 安全
- 默认不保存 Key。
- 勾选保存 Key 才写入当前浏览器 localStorage；公共设备不要勾选。
- 建议使用 server.js 本地代理或自行部署代理，避免浏览器直连失败。
