## 新增重点

- 重新设计颜色层级：左侧彩色导航、模块边界色、卡片分类色、可选高光色。
- 新增彩色 RSS 知识流：语法、词汇、搭配、热点均可展开。

- 热点情景扩充：80 个情景卡，覆盖 AI、航天火星、脑机接口、生物安全、海洋、塑料污染、垃圾分类、气候能源、城市社会等。
- 长难句实验室：逐层点击主干、插入语、逻辑、译序。
- 阅读增强：词首加粗、辅音加粗、聚焦灯、字体大小、行距、纸质/磨砂/深色模式。
- 手机端：底部横向导航，避免左栏占屏。

## 运行方式

### 直接打开

双击 `index.html` 即可使用。AI 浏览器直连可能受到 CORS 限制。

### 本地代理方式

```bash
npm install
# Windows PowerShell
$env:DEEPSEEK_API_KEY="你的key"
npm start
# 打开 http://localhost:8787
```

前端不会把 Key 写入本地存储。更安全的方式是使用 `server.js`，把 Key 放在本机环境变量中。
